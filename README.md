### Una pequeña app para restaurantes y cafeterias

Esta app nace como proyecto universitario para la asignatura de frameworks web con php y laravel.

## Instrucciones de instalacion

Clone el repositorio usando **git clone**, una vez en la carpeta del proyecto ejecute:
```sh
composer install
```
esto instalara las dependencias del proyecto de laravel.

El proyecto hace uso tambien de paquetes de nodejs como:
- Vite
- Alpinejs
- Axios
- Tailwindcss
- Bootstrap Icons

asi que tedra que instalar las dependencias usando:
```sh
npm install
```
